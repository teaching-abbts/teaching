# HTML-Tabellen 1

- Mit **HTML-Tabellen** können Daten in **Zeilen** und **Spalten** dargestellt werden
- Mit `<tr>` wird eine **Zeile** *(table row)* definiert
- Mit `<td>` wird eine **Zelle** *(table data)* definiert. Ein `<th>` ist eine Kopf-Zelle *(table heading)*
