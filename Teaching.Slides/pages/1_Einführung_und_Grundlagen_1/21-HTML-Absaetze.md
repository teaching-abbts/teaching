# HTML-Absätze

<v-clicks>

HTML-Absätze werden mit dem Tag `<p>` definiert.

</v-clicks>

::right::

```html {monaco-run}
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>

  <body>
    <p>Absatz 1</p>
    <p>Absatz 2</p>
    <p>Absatz 3</p>
  </body>
</html>
```
